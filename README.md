## :zap: Getting started
1. Clone this repo using: `https://github.com/lucasvss2/Github-Search`
2. Move yourself to the appropriate directory: `cd gitsearch`
3. To install all dependencies, run: `yarn`
4. Run `yarn start` to start the website
